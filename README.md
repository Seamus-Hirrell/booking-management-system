## Prerequisites

Running this project requires you to install [node](https://nodejs.org/en) and [pnpm](https://pnpm.io/).

## Development

To start a local development server, run:

```shell
pnpm start
```

## Production

To generate a build, run:

```shell
pnpm build
```

## Testing

To run tests, run:

```shell
pnpm test.unit
```
